## üöÄ Getting Started

### Backend (Spring Boot)

```bash
cd backend
./mvnw spring-boot:run
```

Make sure PostgreSQL is running and `application.properties` is configured.

### Frontend (React Native with Expo)

```bash
cd frontend
npm install
npx expo start
```

Open in Expo Go or run on an Android emulator.
